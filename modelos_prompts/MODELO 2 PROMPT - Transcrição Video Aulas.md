# PROMPT UNIVERSAL PARA TRANSCRIÇÃO E ORGANIZAÇÃO DE VÍDEOS - MODELO 2

---

## INSTRUÇÕES INICIAIS

Você vai transcrever e estruturar um vídeo seguindo exatamente este modelo.

### Princípios obrigatórios:

- A estrutura abaixo é **fixa**: não pode ser removida, adicionada ou alterada.
- O **conteúdo interno** deve se ajustar ao tema real do vídeo.
- **Nada pode ser inventado** ou interpretado — apenas o que foi dito.
- A **ordem e o estilo** do professor devem ser preservados.
- A transcrição deve ser **clara, organizada e fácil de consultar**.

### Funciona para qualquer tipo de vídeo:

✓ Marketing  
✓ Negócios  
✓ Técnicas  
✓ Saúde  
✓ Tecnologia  
✓ Formação profissional  
✓ Qualquer outro tema

## INSTRUÇÕES FINAIS

### Ao executar a transcrição, siga estas regras:

1. **Fidelidade total** — Nada de interpretações, adições ou conteúdo externo.
2. **Sem adições externas** — Apenas o que foi dito no vídeo.
3. **Ordem original preservada** — Mantenha a sequência lógica do vídeo.
4. **Tom e estilo mantidos** — Preserve a linguagem do professor.
5. **Exemplos sempre incluídos** — Nenhum exemplo pode ser omitido.
6. **Seções sempre completas** — Todas as 14 seções devem estar preenchidas.
7. **Clareza máxima** — Cada seção deve ser autoexplicativa.
8. **Nada de repetições desnecessárias** — Evite redundâncias.
9. **Citações literais apenas quando ditas** — Não parafrasear citações.
10. **Formatação consistente e limpa** — Use Markdown puro.
